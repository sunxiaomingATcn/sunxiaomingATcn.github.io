自动轮播

使用方法

var carousel = new Carousel(
 {
  images: [
   'images/carousel/bg1.jpg',
   'images/carousel/bg2.jpg',
   'images/carousel/bg3.jpg',
   'images/carousel/bg4.jpg',
   'images/carousel/bg5.jpg',
  ], //图片数组，建议图片尺寸相同
  constructor: '#page1', //图片容器,建议定义容器大小小于等于图片实际尺寸
  intervalTime: '3000', //切换间隔
  autoAnimate: true //是否自动播放
 }
carousel.initLoadImage()//轮播实例初始化
)

点击事件触发轮播
